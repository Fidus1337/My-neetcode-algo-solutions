My main goal is not just to get an "Accepted" status, but to find the most optimal solution in terms of **Time Complexity** and **Space Complexity**, while keeping the code clean, readable, and well-documented.

## 📂 Repository Structure

The problems are grouped by topic following the original NeetCode roadmap. Each file includes:
* A link to the original problem.
* A detailed explanation of the approach used.
* Time and Space complexity analysis (Big O notation).
* Executable Python code.
